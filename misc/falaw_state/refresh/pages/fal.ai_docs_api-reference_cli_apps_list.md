# fal apps list

```bash theme={null}
Usage: fal apps list [-h] [--debug] [--pdb] [--cprofile] [--env ENV]

List applications.

Options:
  -h, --help  show this help message and exit
  --env ENV   Target environment (defaults to main).

Examples:
  fal apps list
  fal apps list --env staging
```
