falaw.bridges.mcp

MCP bridge (planned): expose falaw tools as MCP tools.

The plan: walk falaw.registry.list_tools() and emit one MCP tool per ToolSpec, using the input/output schemas already on the spec. Until then, agents can use the Python tools directly via the Claude skill.
