falaw.operations.llm

LLM-driven tools: text completion, screenplay → Scene IR, IR editing.

These wrap fal-ai/any-llm so the agent (or the user) can invoke an LLM inline as part of a falaw workflow — e.g. parsing a treatment into a structured Scene, or rewriting a Beat from a directorial note.

We intentionally keep these thin. The contract is: any-llm is the universal LLM endpoint, you pick the model via model, and we return plain Python (str or parsed dict).
